This component allows conditional rendering of children components.

If the value defined in the condition attribute is evaluated to true, webc-if will render its children accordingly.

The general use of webc-if is the following:

CONTENT_TO_SHOW_WHEN_CONDITION_IS_TRUE CONTENT_TO_SHOW_WHEN_CONDITION_IS_FALSE

Use with true slot only

CONTENT_TO_SHOW_WHEN_CONDITION_IS_TRUE

Use with without slots

CONTENT_TO_SHOW_WHEN_CONDITION_IS_TRUE

Observations: container can be any HTML tag or custom component both true and false slots are optional. if neither true nor false slots are specified, then the whole webc-if content will be considered the true slot. Under this consideration, the two above examples are identical functional-wise.

In this example the webc-if component is bound to the value of the gdpr model property and it will render the paragraph only if the property is true.

// Controller const { WebcController } = WebCardinal.controllers; const model = { gdpr: 0, }; export default class WebcTestConditionController extends WebcController { constructor(element) { super(element); this.model = this.setModel(JSON.parse(JSON.stringify(model))); this.onTagClick("toggle-property", () => { this.model.gdpr = !this.model.gdpr; }); } }

This example will demonstrate how to use the webc-if together with a model expression. An in-depth explanation of model expressions can be found here.

For this case, a model expression is used to validate the value of an email text input and show an error message if validation fails.

As seen in the previous code samples, the webc-if needs to be placed inside a controller aware component in order to have access to the model so it can evaluate either a property or a predefined expression. Click here to learn more about model expressions. Regardless of the type of condition used, the attribute value must start with the @ character, as explained in the Cardinal Model chapter.

webc-if supports the if/else construct by using slots. To enable this behavior the children of the component must name the slot for each outcome of the condition. This component supports the following slots:

true false

Consider the code example above, to display a certain element when the condition is true, set the slot attribute to the true value, use false otherwise.
