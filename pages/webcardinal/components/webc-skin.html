This component has the power to style generated elements inside any ShadowRoot.

webc-skin is an alternative to CSS ::part for components which are not using this attribute.

 Hello World! custom-shadow-component[data-shadow] .custom-item > selection { text-decoration: underline; }
Explanation

Assuming that the fragment generated by the custom-shadow-component is:

#shadoow-root (open)
Custom selection #text reveal
Hello World!

The expected result is that both "Custom selection" and "Hello World!" are bold.

In #shadow-root fragment, a style tag marked with attribute data-styled containing desired CSS should be appended.

Component webc-skin must be used only when it is necessary!

If styling is applied to an element outside of ShadowRoot fragment a warning will be issued.

Resolve that warning be replacing webc-skin with a link (stylesheet) or a style element.
In your CSS code, shadow root entrance must be marked with [data-shadow];
Multiple selectors can not be used;
Using @keyframes, @import and other "At-rules" will not work properly.
