This web component is an UI element. Navigation is done through TABs similar to those in the browser.

Each tab is a component called psk-tab and the wrapper component is the present one, named psk-tab-navigator.

It should be noted that the tab visibility control area is re-rendered when changing tabs. The actual tab which is visible uses a slot mechanism.

Feel free to inspect the working mechanism of this component in DevTools.
