This component creates a psk-chapter using the title property.

It's role is to create a good looking description for the documentation.

If the title property is not given than the component will create a psk-card instead of a psk-chapter.

This component has one slot as a HTML child in which the data between tags will be rendered inside the:

psk-chapter(if the title is given) or psk-card(if it is not given)

Minus 40 degrees Celsius is exactly the same as minus 40 degrees Fahrenheit.

A lion's roar can be heard from eight kilometres away.

Cats can hear ultrasound.
