This component handles the loading and running of a SSApp , more information on how to create a SSApp you have on the How to create a SSApp page.

It loads the SSApp inside an iframe using the properties given.

In PrivateSky a SSApp is the equivalent of the Decentralised Applications (DApps).

In order to better understand what is a SSApp and it's role in PrivateSky you have all the info you need here.

This component has no HTML Children.
