This component will render a list of items provided through view-model and then will provide a order change mechanism.

Every time the order of the items change, the model that was provided will also update.
items

Represents a list of DraggableOption

label: string

value: string

selected?: boolean

This component has no HTML children.

 import ContainerController from "../../cardinal/controllers/base-controllers/ContainerController.js"; const model = { draggableListModel: { items: [ {label: "Romanian", value: "ro", selected: true}, {label: "English", value: "en"}, {label: "German", value: "de"} ] } } export default class DraggableListExampleController extends ContainerController { constructor(element) { super(element); this.model = this.setModel(model); this.model.onChange('draggableListModel.items', () => { console.log('Order of the list changed', this.model.draggableListModel) }); }; }
