The scope of the component is to allow the user to upload and to display an image.

The srcproperty uses the relative path to that image.

When the user uploads an image, an event is triggered with the content of the image sent as ArrayBuffer.

The contents of the image will be available after catching the event inside a controller.

This component accepts no HTML children or web components.

 import ContainerController from "../../cardinal/controllers/base-controllers/ContainerController.js"; export default class ImgInputController extends ContainerController { constructor(element) { super(element); this.model = this.setModel({ placeholder: "Click to choose an image", alt: "image input component", label: "Image Input Component", eventName: "img-input" }); this.on("img-input", (event) => { this.image = event.data; // Here the user can process the uploaded image this.model.setChainValue("src", this.image); }); } }
