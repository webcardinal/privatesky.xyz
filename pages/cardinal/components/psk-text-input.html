This component is extended from the psk-input component, it is used inside a psk-form component and is intending to be used as input for simple texts, like name, address and so on.

The other components derived from psk-input are:

psk-password-input psk-email-input psk-date-input psk-number-input import ContainerController from "../../cardinal/controllers/base-controllers/ContainerController.js"; const model = { name: { label: "Name", name: "name", required: true, placeholder: "Name here...", value: '' }, email: { label: "Email ", name: "email", required: true, placeholder: "Email address here...", value: '' }, password: { label: "Password", name: "password", required: true, placeholder: "Password here...", value: '' } } export default class FormComponentsExampleController extends ContainerController { constructor(element) { super(element); this.model = this.setModel(JSON.parse(JSON.stringify(model))); } }

A very good example of using this component and the other form components can be found here: Using forms.
