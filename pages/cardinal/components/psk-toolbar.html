This component creates a toolbar using the given properties and the HTML children.

If the property icons is passed ,then a pool of psk-icon, else a pool of button will be instantiated.

Each time a pool item is clicked a new event is created with the eventData property passed along.

This component consumes properties inside the HTML children in this way:

  • If the slot has the same name as one of the actions, then a new slot with that name is created.
  • The icon sent as property will be rendered just like a normal psk-icon.
