This component role is copying in the clipboard the web address of a selected text zone using the properties of this component.

It is easy to use as it gets the id as property so it can be easily copied and send forward.

It is rendered inside the psk-card so the user can benefit of it's usefulness for every card that is created.

This component has a slot tag so the data you enter between the tags can be rendered as the copy button.

Click on the # element in the top right corner in order to copy the address in the clipboard.
