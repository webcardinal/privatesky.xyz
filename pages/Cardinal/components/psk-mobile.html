This component is a wrapper for any page of a mobile application.

It must be used like psk-page but only for mobile devices.

This component has two asynchronous methods called toggleSidebar and toggleOptions.

If they are called without any parameters they will change the current state of the respective element. For example, if the options menu is visible and the toggleOptions() function is called, it will become hidden, after a second call the menu will become visible again.

If another behavior is desired, the functions accept a boolean value which represents the visibility of the element: true only show sidebar / options menu, false hide that element.

There are three slots available for this component named "title", "footer" and "options".

The developer has the possibility to use a custom title.

If a slot name "title" is given, the title property from component is ignored.
